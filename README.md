# Navigation

Projeto desenvolvido no Android Studio para demonstrar navegação entre telas com Jetpack Compose.

## Funcionalidades
- Tela Home
- Tela de Detalhes
- Navegação entre telas
- Botão Voltar

## Tecnologias
- Kotlin
- Android Studio
- Jetpack Compose
- Navigation Compose
